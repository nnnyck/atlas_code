Como adicionar Css ao código

Quando um Navegador lê a cascata de estilo, ele formata o documento HTML de acordo com os estilos definidos pelo CSS.

São 3 Formas de Adicionar o CSS

As 3 formas de adicionar o css são:

  • Css Externo
  • Css Interno
  • Css em Linha

Css Externo

É a forma mais recomendada de se aplicar CSS. Essa forma ajuda a separar, deixa o código mais limpo e de fácil manutenção.

Cada documento HTML deve conter a referência para o arquivo .css, que é dado através da tag <link>

Ordem de Cascata

Os estilos css tem uma certa ordem de prioridade que é interpretada pelo navegador.

  1. Css em Linha
  2. Css Interno
  3. Css Externo

  4. Css Padrão do navegador

Assim o código css com a maior prioridade (1) irá sobrepor os de menor prioridade (2,3 e 4).
