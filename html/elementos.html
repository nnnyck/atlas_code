Elementos HTML

Esta página lista de forma organizada e mais fácil de compreender os elementos html que fui aprendendo.


Elementos Básicos

Elemento Descrição
DOCTYPE O elemento !DOCTYPE é a primeira linha de um código html estruturado. Após o elemento deve-se definir de que tipo ele é, geralmente do tipo html.
<html> O elemento HTML representa a estrutura total do código para o navegador entender como interpretar. Todos os elementos devem estar dentro dessa tag.
<!-- // --> Esse elemento representa um COMENTÁRIO no código html.


Elementos de Composição Básica

Elemento Descrição
<head> O elemento HEAD é onde se configura informações que não aparecem diretamente para o usuário, incluindo seu título, links para scripts e folhas de estilos.
<meta> O elemento META define qualquer informação de metadados que não podem ser definidos por outros elementos HTML.
<link> O elemento LINK especifica as relações entre o documento atual e um recurso externo. Possíveis usos para este elemento incluem a definição de uma estrutura relacional para navegação. Este elemento é mais usado para vincular as folhas de estilo.
<title> O elemente TITLE define o título da página. mostrado na barra de título de um navegador ou na aba da página. Pode conter somente texto e qualquer marcações contidas no texto não são interpretadas.
<body> O elemento BODY representa o conteúdo visível na tela.


Separação de Conteúdo

Elemento Descrição
<header> O elemento HEADER representa o cabeçalho da página. Geralmente, ele contém informações como logotipo, título, navegação e outros elementos relacionados ao início da página ou seção.
<main> O elemento MAIN define o conteúdo principal dentro do body em seu documento ou aplicação. Entende-se como conteúdo principal aquele relacionado diretamente com o tópico central da página ou com a funcionalidade central da aplicação. O mesmo deverá ser único na página, ou seja, dentro do elemento não deverão ser incluidas seções da página que sejam comuns a todo o site ou aplicação, tais como mecanismos de navegação, informações de copyright, logotipo e campos de busca (a não ser, é claro, caso a função principal do documento seja fazer algum tipo de busca).
<section> O elemento SECTION representa uma seção genérica contida em um documento HTML, geralmente com um título, quando não existir um elemento semântico mais específico para representá-lo.
<footer> O elemento FOOTER representa o rodapé da página. Normalmente um rodapé contém informações sobre o autor da seção de dados, direitos autorais ou links para documentos relacionados.


Elementos de Layout

Elemento Descrição
<div> O elemento DIV é um elemento genérico que é frequentemente usado para criar contêineres ou divisões em uma página HTML. Ele é utilizado para agrupar e organizar outros elementos, permitindo uma estruturação mais clara e facilitando a aplicação de estilos CSS.
<nav> O elemento NAV é um elemento específico para marcar a seção de navegação em uma página. Ele é utilizado para agrupar links de navegação, como menus ou barras de navegação. O uso do <nav> ajuda a indicar semanticamente que a seção contém elementos de navegação.


Elementos de Texto

Elemento Descrição
<h1>,<h2>,
<h3>,<h4>,
<h5>,<h6> 		Os elementos H(123456) representam seis niveis de títulos, começando pelo maior (h1) até o menor (h6).
<span> O elemento SPAN é outro elemento html utilizado para agrupar conteúdo em linha e aplicar estilos específicos a partes isoladas de um texto ou outros elementos. Ao contrário do <div>, que é um elemento de bloco, o span é um elemento de linha.
<p> O elemento P representa um parágrafo e é onde são escritos textos.


Formatações de Textos

Elemento Descrição
<strong>,<b> Os elementos STRONG e BOLD representam uma formatação que consiste em deixar as palavras em negrito. Porém a tag BOLD não é a recomendada para uso mais, sendo substituída pela tag STRONG.
<em> O elemento EM rerpresenta uma formatação que consiste em deixar as palavras em itálico.
<u> O elemento U representa uma formatação de texto que consiste em colocar underline nas palavras escritas
<strike> O elemento STRIKE representa uma formatação de texto que consiste em colocar uma linha nas palavras escritas.


Elementos de Experiência do Usuário

Elemento Descrição
<a> O elemento A (âncora) com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo dentro de cada <a> precisará indicar o destino do link.
<br> O elemento BR é principalmente utilizado para criar quebras de linha, mas ele também pode servir como espaçamento vertical.
<hr> O elemento HR representa uma linha horizontal exibida nos navegadores. De forma contextual pode representar uma quebra temática entre elementos de nível de parágrafo (por exemplo , uma mudança da cena de uma história, ou uma mudança de tema com uma seção).
<img> O elemento IMG representa a inserção de imagem no documento. Ele é um elemento vazio, o que significa que não possui uma tag de fechamento.


Listas Ordenadas e Não Ordenadas

Elemento Descrição
<ul> O elemento UL é um elemento pai e é a forma de criar uma lista desordenada. Caracteristicamente, os itens em uma lista desordenada são exibidos com um marcador que pode ter várias formas, como um ponto, um círculo, ou um quadrado.
<ol> O elemento OL é um elemento pai e é a forma de criar uma lista ordenada. De forma característica esses itens ordenados em uma lista são mostrados com uma contagem que os precede, que pode ser de qualquer tipo, como numerais, letras, algarismos romanos, ou simples símbolos.
<li> O elemento LI é usado para representar um item que faz parte de uma lista. Este item deve estar contido em um elemento pai: uma lista ordenada (<ol>) ou uma lista desordenada (<ul>).


Elementos de Tabela

Elemento Descrição
<table> O elemento TABLE é o contêiner principal para toda a tabela.
<tr> O elemento TR representa uma linha dentro da tabela.
<td> O elemento TD define uma célula de dados em uma linha.
<th> O elemento TH efine uma célula de cabeçalho em uma linha. Geralmente, é usado nas primeiras linhas (ou em grupos de linhas) para rotular as colunas.


Formulários

Elemento Descrição
<form> O elemento FORM define o início de um formulário e envolve todos os elementos do formulário.
<input> O elemento INPUT é utilizado para criar campos de entrada de dados, como caixas de texto, botões de rádio, caixas de seleção, etc.
<select> O elemento SELECT cria uma lista suspensa (dropdown) e é geralmente usado em conjunto com <option> para fornecer opções selecionáveis.
<textarea> O elemento TEXTAREA é utilizado para criar campos de texto multi-linha.
<button> O elemento BUTTON cria um botão clicável que pode ser usado para enviar um formulário ou acionar uma ação JavaScript.
<label> O elemento LABEL é usado para associar um rótulo descritivo a um campo de formulário, melhorando a acessibilidade e a usabilidade.
