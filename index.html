Elementos HTML

Esta página lista de forma organizada e mais fácil de compreender os elementos html que fui aprendendo.


Elementos Básicos

Elemento Descrição
DOCTYPE O elemento !DOCTYPE é a primeira linha de um código html estruturado. Após o elemento deve-se definir de que tipo ele é, geralmente do tipo html.
<html> O elemento HTML representa a estrutura total do código para o navegador entender como interpretar. Todos os elementos devem estar dentro dessa tag.


Estrutura

Elemento Descrição
<head> O elemento HEAD é onde se configura informações que não aparecem diretamente para o usuário, incluindo seu título, links para scripts e folhas de estilos.
<meta> O elemento META define qualquer informação de metadados que não podem ser definidos por outros elementos HTML.
<link> O elemento LINK especifica as relações entre o documento atual e um recurso externo. Possíveis usos para este elemento incluem a definição de uma estrutura relacional para navegação. Este elemento é mais usado para vincular as folhas de estilo.
<title> O elemente TITLE define o título da página. mostrado na barra de título de um navegador ou na aba da página. Pode conter somente texto e qualquer marcações contidas no texto não são interpretadas.
<body> O elemento BODY representa o conteúdo visível na tela.
